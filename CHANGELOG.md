# 更新日志

本文档记录 AIPromptWorkbench 的公开版本更新内容。

---

## v1.1.1

**视频模板变量输入、应用内更新与系统交互优化版本。**

该版本主要围绕悬浮窗状态恢复、视频模板变量化编辑、平台专属内容处理、应用内更新、系统托盘和窗口交互进行优化，并修复导出与界面行为问题。

### 新增功能

#### 视频模板可视化变量输入

- 场景详细设置中，引用包含变量的词条后，可切换为可视化变量输入模式。
- 可视化界面会展示当前实际生效内容中的变量、默认值和说明，并允许逐项设置变量值。
- 平台专属内容中的变量仅在对应平台生效时显示。
- 保留旧的特殊格式变量输入方式；该方式仅在当前字段引用有效词条时生效。
- 进入变量输入模式后，隐藏 AI 优化、AI 返回内容及相关复制、覆盖和解析操作，减少重复界面。

#### 应用内自动更新

- 更新公告支持 Markdown 格式渲染。
- 用户确认更新后，软件可自动下载安装器、校验文件大小与 SHA256，并启动安装流程。
- 安装完成后可自动重新启动 AIPromptWorkbench。
- GitHub 与 Gitee Release 中继续保留完整安装器，仍支持手动下载和安装。

#### 系统托盘

- 设置中心新增“关闭窗口”行为选项，可选择“退出程序”或“最小化到系统托盘”，默认使用“退出程序”。
- 软件运行期间始终显示系统托盘图标。
- 左键单击托盘图标可显示并置前主窗口。
- 右键菜单提供“打开主窗口”和“退出”操作。

#### 悬浮窗位置恢复

- 新增“重置悬浮窗位置”功能，可将悬浮窗恢复到当前主屏幕的默认位置和默认展开状态。
- 重置时会解除贴靠、清除异常位置，并恢复默认尺寸。
- 快捷键、设置开关及其他显示入口会保留并恢复悬浮窗原有的展开、收起和贴靠状态。

### 功能调整

#### 变量与词条内容

- 词条正文和平台专属正文中的变量占位符统一使用 `{变量名}`。
- 保留词条引用变量的旧输入格式，以及“变量名 | 默认值 | 介绍”等变量定义格式。
- 可视化变量输入仅展示当前基础内容与所选平台专属内容中实际生效的变量。
- 同名变量只显示一次，用户显式填写的变量值优先。

#### 平台专属内容

- 平台专属内容由“覆盖基础词条内容”调整为“追加到基础词条内容末尾”。
- 平台专属内容中的变量会正常参与预览和导出。

#### 窗口与界面交互

- 下拉选项菜单在主窗口部分位于屏幕外时，首次展开即与对应控件保持横向对齐，避免出现瞬时横向跳动。

### 修复问题

- 修复悬浮窗处于贴靠状态时，通过快捷键或设置开关隐藏后无法正确恢复、贴靠把手不可见的问题。
- 修复平台专属内容可以在引用内容预览中显示，但未参与最终导出的问题。
- 修复未引用词条时，旧特殊输入格式被错误识别为变量输入模式的问题。
- 修复基础词条内容与平台专属内容之间出现多余空白行的问题。
- 修复窗口部分位于屏幕外时，下拉菜单首次展开发生横向错位的问题。

### 移除内容

- 移除外观设置中未实际生效的“界面缩放”选项及相关逻辑。
- 软件界面继续跟随系统显示缩放，不影响高 DPI 和多显示器环境。

### 发布说明

- 本版本为 v1.1.1 优化与问题修复版本。
- 已安装旧版本的用户可通过软件内更新功能下载安装，也可以从 GitHub 或 Gitee Release 页面手动下载安装器。
- 如需校验安装包完整性，请使用 Release 附件中的 `SHA256SUMS.txt`。
- 当前安装器尚未进行代码签名，Windows SmartScreen 可能显示安全提示。

## v1.1.0

**悬浮窗快捷面板与主题体验优化版本。**

该版本重点围绕提示词超市的悬浮使用场景进行增强，使用户可以在其它创作软件前台工作时快速搜索、复制和调用提示词内容，同时修复设置中心与浅色主题下的多处可读性问题。

### 新增功能

#### 悬浮窗提示词快捷面板

- 新增悬浮窗提示词快捷面板，用于快速调用提示词超市词条。
- 支持悬浮窗置顶、无边框、拖动、调整大小和全局快捷键显示/隐藏。
- 悬浮窗仅显示提示词超市内容，每个词条以独立卡片展示标题与简介。
- 支持搜索提示词超市词条。
- 支持沿用提示词超市复制逻辑：
  - 鼠标左键复制正面提示词。
  - 鼠标右键复制负面提示词。
  - 鼠标中键复制完整提示词。
- 支持设置中心开启或关闭悬浮窗功能。
- 支持设置中心配置悬浮窗快捷键。
- 支持设置中心配置自动显隐逻辑。

#### 悬浮窗展开、收起与贴靠

- 新增悬浮窗收起按钮，可隐藏搜索框与提示词卡片区域，仅保留紧凑标题栏。
- 展开与收起时保持窗口宽度不变，并以上边缘为锚点，避免位置跳动。
- 展开状态和收起状态均支持边缘贴靠。
- 靠近屏幕边缘的可贴靠区域时，悬浮窗会自动吸附到屏幕边缘，提示可以触发贴靠。
- 触发贴靠后使用统一的边缘把手样式，不区分展开来源或收起来源。
- 鼠标移入贴靠把手后自动展示完整悬浮窗，并恢复贴靠前的展开或收起状态。
- 鼠标离开展示出的悬浮窗后快速重新贴靠隐藏。
- 贴靠隐藏与展示过程加入过渡动画，减少直接切换造成的突兀感。

### 发布说明

- 本版本为 v1.1.0 功能更新版本，重点提升提示词快捷调用效率和悬浮窗稳定性。
- 建议从官方 GitHub 或 Gitee Release 页面下载安装器，并核对 `SHA256SUMS.txt`。
- 当前安装器尚未进行代码签名，Windows SmartScreen 可能显示安全提示。

## v1.0.0

**首个稳定公开版本。**

该版本完成了 AIPromptWorkbench 的基础产品形态，建立了提示词资产管理、视频模板编辑、Git 提示词仓库同步、安装器发布和自动更新清单等核心能力。

### 新增功能

#### 提示词资产管理

- 支持普通提示词词条管理，用于沉淀风格提示词、角色设定、场景描述、镜头语言、负面提示词等可复用内容。
- 支持 AI 系统提示词方案管理，用于保存完整的任务型提示词规则和系统提示词配置。
- 支持结构化提示词文件，便于提示词分类、维护、分发和版本化。

#### 视频模板与场景详细编辑

- 支持面向 AI 视频生成的提示词模板组织方式。
- 支持将视频提示词拆分为多个可编辑部分，便于分别控制主体、场景、动作、镜头、光影、风格和质量要求。
- 支持整体提示词预览，方便在编辑过程中检查最终组合效果。
- 支持变量化内容，提升同类模板在不同角色、地点、镜头和风格之间的复用效率。

#### Git 提示词仓库支持

- 支持从 Git 仓库获取提示词资源。
- 支持将远程仓库提示词与本地提示词分目录存储，避免远程资源和本地私有资产混淆。
- 支持公开提示词仓库的索引与分发流程。

官方自带提示词仓库：

```text
https://gitee.com/WeepingLeaf/AI-Prompt-Repository.git
```

#### 可见性、许可与加密策略

- 支持围绕提示词资产设置可见性方案。
- 支持公开元数据、私有正文、完全公开、完全私有等不同分发思路。
- 支持为提示词资产附带许可信息。
- 支持面向受限分发场景的加密设计，为团队或私有仓库使用提供基础能力。

#### 安装器与自动更新

- 新增 Windows 安装器发布流程。
- 新增 `latest.json` 更新清单，用于应用内版本检查。
- 新增 `SHA256SUMS.txt`，用于发布文件校验。
- 新增公开发布仓库结构，用于承载安装器、更新日志和版本元数据。
- 支持 GitHub 与 Gitee 双发布源。

### 发布说明

- 当前安装器尚未进行代码签名，Windows SmartScreen 可能显示安全提示。
- 建议从官方 GitHub 或 Gitee Release 页面下载安装器，并核对 `SHA256SUMS.txt`。
- 本公开仓库不包含完整源代码，仅用于发布安装器、更新清单、校验文件和版本说明。

### 当前版本定位

v1.0.0 是 AIPromptWorkbench 的第一个稳定发布版本，重点解决以下问题：

- 将零散提示词整理为可维护的结构化资产。
- 将复杂视频生成提示词拆分为可编辑、可复用的模板。
- 将本地提示词与远程提示词仓库分离管理。
- 为后续提示词仓库、模板市场、团队共享和自动更新奠定基础。
